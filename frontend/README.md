# Frontend - Team Task Manager UI

React + Vite client for the Team Task Manager application.

## Requirements

- Node.js (LTS recommended)
- npm

## Setup

1. Install dependencies:
   - `npm install`
2. Start development server:
   - `npm run dev`

Default frontend URL: `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start Vite development server
- `npm run build` - Build app for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Backend Connection

API base URL is configured in `src/api/client.js`:
- `http://localhost:5000/api`

Make sure the backend server is running before using authenticated or data-driven pages.
